# 命令行的一些常见命令

> 在使用命令行的时候使用快捷键能够帮助我们更快的输入命令。

## 移动

```bash
# 移动到行尾 (end)
Ctrl + e
# 移动到行首 (ahead)
Ctrl + a
```

## 删除

```bash
# 删除光标前一个单词
Ctrl + w
# 删除光标后到行尾
Ctrl + k
# 删除命令行所有文本
Ctrl + u
# 向行首删除一个字符
Ctrl + h
# 向行尾删除一个字符
Ctrl + d
```

## 粘贴

```bash
# 粘贴Ctrl+u，Ctrl+k，Ctrl+w删除的文本
Ctrl + y
```

## 后台运行

```bash
Ctrl + z

fg 恢复
```
