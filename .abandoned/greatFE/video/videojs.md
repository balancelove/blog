# 视频播放器

## videoJS

- 自定义 UI，表现一致，可能有限制，如果你不喜欢的话
- 丰富的插件
- 兼容性非常好
- 使用简单易上手
- 开源免费

GitHub 地址： https://github.com/videojs/video.js

## hls.js

- 适合做 hls 的直播、点播
- 小巧，只做直播，可以更好的自己定制化ui

GitHub 地址： https://github.com/video-dev/hls.js

## flv.js

- B 站开源的 flv 格式的播放器

GitHub 地址： https://github.com/Bilibili/flv.js

## videojs 直播

`videojs-contrib-hls` 直播插件。
