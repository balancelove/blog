# 浏览器视频特效

## 基本概念

- CSS filter, -webkit-filter, filter
- video 关联 filter
- OpenGL、Metal

## 支持的种类

|    特效     |   说明   |
| :---------: | :------: |
|  grayscale  |   灰度   |
|    sepia    |   褐色   |
|  saturate   |  饱和度  |
| hue-rotate  | 色相旋转 |
|   invert    |   反色   |
|   opacity   |  透明度  |
|    blur     |   模糊   |
|  contrast   |  对比度  |
| drop-shadow |   阴影   |
| brightness  |   亮度   |
